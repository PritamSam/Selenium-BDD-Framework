# Selenium-BDD-Framework
This repository describes Selenium BDD Automation Framework

To run this project, right click on TestRunner.java and run as TestNG Test

After running the full test suite, refresh the Reports root folder by right-clicking on it and selecting Refresh. 
Then, within the Reports folder, right-click on the first report file and select Open With > Web Browser.

Project Overflow -

```
pom.xml
   │
   ▼
TestRunner (Execution Entry Point)
   │
   ▼
Driver Layer
   │
   ▼
Hooks (Base Layer Replacement)
   │
   ▼
Feature Files (BDD Specs)
   │
   ▼
Step Definitions (Test Logic Layer)
   │
   ▼
Page Objects (UI Layer)
   │
   ▼
Utilities (Support Layer)
   │
   ▼
Listeners (Reporting Layer)
   │
   ▼
Reports + Logs + Screenshots
```
